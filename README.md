# Caderno de Exercícios — Bits e Processadores

Este repositório contém um caderno de exercícios desenvolvido para a disciplina Bits e Processadores, com foco em compreensão conceitual de sistemas digitais, arquitetura de computadores e linguagem de máquina.

## Conteúdo
- Álgebra Booleana
- Dados Digitais e bases numéricas
- Sistemas Síncronos e Flip-Flops
- Aritmética Binária e Somadores
- Arquitetura da CPU Z01
- Assembly e Linguagem de Máquina
- Gabarito comentado por tema

## Objetivo
O objetivo deste material é consolidar o entendimento dos mecanismos internos do computador, indo além da resolução mecânica de exercícios, com explicações e diagramas que evidenciam o raciocínio por trás das soluções.

## Organização
- `exercicios/`: enunciados organizados por tema  
- `gabaritos/`: resoluções comentadas  
- `main.tex`: arquivo principal para compilação do PDF  

## Compilação
O documento pode ser compilado com qualquer distribuição LaTeX padrão:

```bash
pdflatex main.tex
